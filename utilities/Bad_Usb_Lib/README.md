This library provides USB HID / keyboard layout support and must be placed in the Arduino ESP32 USB libraries folder before compiling firmware that uses BadUSB features.

Windows:

`C:\Users\<username>\AppData\Local\Arduino15\packages\m5stack\hardware\esp32\2.1.1\libraries\USB\src`

Linux:

`~/.arduino15/packages/m5stack/hardware/esp32/2.1.1/libraries/USB/src`
